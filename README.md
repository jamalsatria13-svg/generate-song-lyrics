# generate-song-lyrics
Display song lyrics by terminal

# file descriptions
- converter.py: converts the .irc format to match the program's required format
- liriklagu1.py: for songs without translations
- liriklagu2.py: for songs with translations

# how to use liriklagu2.py
- download the files and place them in a new folder
- add your lyrics.irc file and your song to the same folder
- use converter.py to convert the lyrics.irc format
- paste the output into the liriklagu.py code
- run the program (liriklagu1.py or liriklagu2.py)
- press "Enter" to start
- press "Ctrl + C" to stop
